# BioPyLab v3.0

BioPyLab is a professional bioinformatics desktop application developed in Python and PySide6.

## Features

- DNA Sequence Analysis
- ORF Finder
- Sequence Alignment
- Mutation Detection
- Protein Analysis
- K-mer Analysis
- Restriction Enzyme Analysis
- Data Export (PDF, Excel, CSV)

## Technologies

- Python
- PySide6
- Matplotlib
- ReportLab
- OpenPyXL
